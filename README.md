# simple-download-cli

![placeholder][img_url]
> Cli version of simple download.

## Installation
To install **simple-download-cli** you need 
to use a **node package manager** as **npm** 
or **yarn**

```bash
    $ npm install simple-download-cli --global
    
    # or with shorthands
    
    $npm i -g simple-download-cli 
```

## Usage
```bash
 $ download --help
   Usage
    download <url> [filename]

    -p --path <path>
```
